# הצ'אטבוט של שני - מדריך פריסה

הצ'אטבוט מוכן. הנה 4 צעדים להפעיל אותו בעולם (סה"כ 15-20 דקות).

---

## צעד 1: קבלי מפתח Gemini API (5 דקות, חינם)

1. היכנסי ל-[aistudio.google.com](https://aistudio.google.com)
2. התחברי עם חשבון Google (shanishlomo211@gmail.com)
3. אשרי את תנאי השימוש
4. בצד ימין למעלה, לחצי על **"Get API key"**
5. לחצי **"Create API key"** > בחרי פרויקט חדש
6. **העתיקי את המפתח** (מחרוזת ארוכה שמתחילה ב-`AIza...`) ושמרי אותו במקום בטוח

⚠️ אל תוסיפי כרטיס אשראי. אנחנו נשארים במסלול החינמי.

---

## צעד 2: פתחי חשבון GitHub (5 דקות, חינם)

הבוט צריך להיות ב-GitHub כדי ש-Vercel יוכל לפרוס אותו.

1. היכנסי ל-[github.com](https://github.com) והירשמי עם המייל שלך
2. אשרי את המייל
3. לחצי על **New Repository** (הכפתור הירוק)
4. שם: `shani-chat`
5. הגדירי **Public**
6. סמני V ליד **Add README**
7. לחצי **Create Repository**

---

## צעד 3: העלי את קבצי הבוט ל-GitHub (3 דקות)

1. בתוך הריפוזיטרי החדש, לחצי **Add file → Upload files**
2. גררי לתוך שלושת הקבצים והתיקייה שקיבלת ממני:
   - `index.html`
   - `api/chat.js` (כן, גם התיקייה `api`)
   - `package.json`
   - `vercel.json`
3. בתחתית, לחצי **Commit changes**

---

## צעד 4: פרסי דרך Vercel (5 דקות, חינם)

1. היכנסי ל-[vercel.com](https://vercel.com)
2. **Sign up with GitHub** (התחברי דרך החשבון שיצרת עכשיו)
3. במסך הראשי, לחצי **Add New → Project**
4. תראי את הריפוזיטרי `shani-chat` שיצרת. לחצי **Import**
5. במסך הגדרות, לחצי על **Environment Variables** ופתחי:
   - **Name**: `GEMINI_API_KEY`
   - **Value**: הדביקי את המפתח שהעתקת מ-Google AI Studio (המחרוזת שמתחילה ב-`AIza...`)
   - לחצי **Add**
6. לחצי **Deploy**

## מה עושים אם משהו לא עובד?

**"Function timeout" או "Error 500"**  
בדקי ב-Vercel > Settings > Environment Variables שהמפתח `GEMINI_API_KEY` הוזן נכון.

**הבוט לא עונה בכלל**  
לחצי F12 בדפדפן, פתחי את Console, ותראי הודעת שגיאה. שלחי לי צילום מסך.

**רוצה לתקן משהו בטקסטים / פתיחה**  
זה עריכה של `index.html` (לפתיחה) או `api/chat.js` (לפרומפט המערכת). שלחי לי מה לתקן ואני שולח לך גרסה חדשה.
